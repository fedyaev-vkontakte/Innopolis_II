# Innopolis_II
## Моделирование зависимости роста спортсмена от его веса, возраста, пола и вида спорта
### Описание проекта
### Цель
Построить и сравнить несколько моделей машинного обучения для предсказания роста (Height) спортсменов на основе их веса, возраста, пола, вида спорта и сезона Олимпийских игр.

### Задачи
1. Загрузить и изучить данные об олимпийских атлетах с 1896 по 2016 год;

2. Очистить данные — обработать пропуски, выбросы, неинформативные колонки;

3. Преобразовать категориальные признаки в числовой формат;

4. Обучить 5 моделей регрессии:

     - K-ближайших соседей (KNN);

     - Линейная регрессия;

     - Дерево решений;

     - Случайный лес;

     - Градиентный бустинг;

5. Оценить качество моделей по метрикам MAE, MSE, R²

6. Выбрать лучшую модель на основе совокупности метрик

### Данные
- Источник: athlete_events.csv — исторические данные об Олимпийских играх

- Объем: 271 116 записей, 15 столбцов

- Признаки после обработки:
     - Age (возраст);

     - Weight (вес);

     - Sex_M / Sex_F (пол — one-hot);

     - Season_Summer / Season_Winter (сезон — one-hot);

     - Sport (вид спорта — one-hot, 66 уникальных значений).
 
### Целевая переменная
 - Height (рост в см).

## Резюме
### Проект демонстрирует полный пайплайн Data Science:

 - Загрузка и EDA;

 - Очистка и feature engineering;

 - Обучение 5 моделей регрессии;

 - Оценка и выбор лучшей модели.

### Кому и зачем может понадобиться такая модель:
 Модель позволяет спортивным академиям отбирать детей с подходящим прогнозом роста под конкретные виды спорта (баскетбол, волейбол, гимнастика), сокращая затраты на подготовку заведомо неподходящих кандидатов.
